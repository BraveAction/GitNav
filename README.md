# GitNav

 - [Picasso][picasso]
 - [CircleImageView][circle]
 - [EasyFonts][easy]
 - [OkHttp][okhttp]
 - [Gson][gson]
 - [PrettyTime][pretty]
 - [Butterknife][butter]
 - [MagicIndicator][magic]
 - [MarkdownView-Android][markdown]
 - [RxAndroid][rxandroid]
 - [Egit-GitHub][egit]
